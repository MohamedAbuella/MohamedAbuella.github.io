Using pandapower -an open-source python tool- for building and analysis the 5-bus test system from Glover et al. [1], and then making any changes on the system and observe the impacts of these changes by run the power flow solver on the pandapower.

The diagram and the data of this 5 bus test power system can also be found online by clicking on this Link

The sharable Jupyter notebook of the base case of the system is available on GitHub Repo

Whereas, the notebook for another case of adding shunt capacitor banks on Bus-2 is available on GitHub Repo


[1] J. D. Glover, M. S. Sarma, and T. J. Overbye, Power System Analysis and Design, 5th ed. Stamford, CT: Cengage Learning, 2012.
